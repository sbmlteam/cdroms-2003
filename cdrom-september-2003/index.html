This CD-ROM contains a collection of documents, software and other files about the Systems Biology Markup Language (SBML), a model definition language for representing biochemical reaction networks in a machine-readable format. SBML is based upon XML, which in turn is a simple and portable text-based medium that is widely used in computational biology and bioinformatics.

The following article describes SBML Level 1. Click on the link below to obtain a copy from the publisher. (We regret we are not permitted by the publisher to redistribute the article in electronic form.)

The Systems Biology Markup Language (SBML): A medium for representation and exchange of biochemical network models

software Numerous software packages featuring SBML support
This folder contains a number of software packages having the ability to read and write models in SBML format. Many of these packages were developed and are maintained by third parties, not the ERATO Kitano Symbiotic Systems project. Please see the individual software folders for more information about their authors, terms of distribution, instructions for use, and other information.
  • CellDesigner, a graphical SBML model editor written in Java
  • Jarnac, a powerful ODE-based biochemical network simulator for Microsoft Windows
  • JDesigner, an advanced, full-featured model editor for Microsoft Windows
  • libsbml, a C/C++ library for reading, writing and manipulating SBML
  • MathSBML, a Mathematica package for simulating, visualizing and translating SBML models
  • SBW, a portable application integration framework

This collection of software is only a small fraction of the software packages known to support SBML today. At the time of this writing, over 30 software packages can read and/or write SBML, and more are being developed. Please visit the project website, http://www.sbml.org, for a more complete list.
sample-models SBML files containing sample models
Included on this CD-ROM is a collection of models in SBML format. The collection consists of about 15 models drawn from published papers in the literature, and the SBML Test Suite, a set of over 9000 model files in SBML Level 2 and Level 1 format useful for software developers for testing their software.

mailing-lists Archives of electronic mailing lists on the topic of SBML
This folder contains the archives of messages sent to the various SBML mailing lists up to the time of creation of this CD-ROM (29 August 2003). The archive is split into two subfolders:
  • Archive of the sbml-discuss@caltech.edu mailing list since the list's creation on 30 August 2002.
  • Archive of messages sent to mailing lists that existed prior to the creation of sbml-discuss:
    • sbml-arrays
    • sbml-diagrams
    • sbml-geometry
    • sbml-metadata
    • sbml-modularity
    • sbml-multistate
    • sbml-other

Historical note: The use of multiple separate mailing lists turned out to be an unsuccessful experiment in SBML community organization, and after some time, the SBML community adopted the single sbml-discuss list that is now in existence. sbml-discuss@caltech.edu is today the primary discussion forum for the SBML community. To be added to the list, please email your request to sbml-team@caltech.edu.
