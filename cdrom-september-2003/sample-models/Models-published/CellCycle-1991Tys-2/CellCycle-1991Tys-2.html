Cell Cycle Model - Tyson (1991, 2 variables)

This is a two variable reduction of the larger 6- variable model published in the same paper. The published equations are

du/dt = k4 * (v - u) * (alpha + u2) - k6 * u

dv/dt = Kappa - k6 * u

In the present implementation, the change of variables z = v - u is made, so that the system becomes the more symmetric system

du/dt =  k4 * z * (alpha + u2) - k6 * u

dz/dt = Kappa - z * (alpha + u2)

The variables can be interpreted as follows:

u = [activeMPF] / [CT]

v = ([cyclin] + [preMPF] + [activeMPF]) / [CT]

z = ([ cyclin] + [preMPF]) / [CT]

where [CT] = [CDC2] + {CDC2P] + [preMPF] + [aMPF]
