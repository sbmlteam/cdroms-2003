Herbert Sauro wrote:
> 
> Question to the community: Does any simulator use the local namespace in a
> truely local namespace fashion?
> 

Yes, I use it in my SBML import/export for connecting to E-cell 1/2.

To me it makes a lot of sense to be able to use parameters such as Vm 
and Km repeatedly, since they have certain meaning for specific 
mechanisms (and I can resue the same formula for each mechanism).

I also have a first prototype of a translator capable of converting an 
SBML L1 model with local parameters into one with only global parameters 
(and vice versa). It is not ready for distribution yet, but if anyone 
would need such a tool, please contact me, as that would be a motivation 
to finish it.

Joerg.
