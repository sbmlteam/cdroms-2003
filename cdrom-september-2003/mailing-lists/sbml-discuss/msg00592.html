Hi Jeremy,

It's great to hear about this progress -- bravo!

As far as L2-compliant software goes, Bruce Shapiro's
MathSBML is a Mathematica package for working with SBML and
supports both Level 1 and Level 2.  It supports very nearly
everything in SBML Level 2, including such things as
Algebraic Rules.

You can download it from http://sourceforge.net/projects/sbml
(When the new sbml.org is up  -- a preview of which you saw
during the Hackathon -- we will have a better set of web
pages for MathSBML and libSBML.)

If you do try it out, please make sure to send Bruce your
feedback.
