Hello there,

  For those of you who attended the SBML Hackathon: thanks,  I had a great
time, and actually got something accomplished, too...which is why I'm
emailing this list.  I've actually managed to successfully
generate valid SBML level 2 files from Biocyc, at least according
to the libSBML version 2.0.1 validator.  What I was wondering is if there is
any level 2 compliant software I could try to load these files into.

As these are steady-state models, and contain no kinetic parameters, any
software that can represent networks and stoichiometries would be useful.
Ideally, I'd like to use JDesigner to represent the nodes, and then run
SBWMetatool
to find the extreme pathways and elementary flux modes.
However, JDesigner 1.8k keeps coughing up blood whenever I try to load the
files.
It complains that there is an "Error in ParseXMLDoc:  Badly formed SBML: no
reactants specified in reaction
[ATP-SYN-RXN]".

ATP-SYN-RXN does have reactants, but it is the first reaction in
<listOfReactions>. If I remove that reaction from the SBML file,
then JDesigner complains that the next reaction has no reactants.

I tried installing the other SBW software modules, Gepasy, and the Biospice
agents, without succcess.
The closest I got was the SBW SBMLValidator, which did validate my files,
but
wasn't very interesting otherwise.

Anyway, if you go to http://genome.dfci.harvard.edu/~zucker/BPHYS/sbml.zip

you can download all the models I have generated from the Biocyc
Pathway/Genome databases,
and test them on your SBML level 2 compliant software.

Sincerely,

Jeremy
