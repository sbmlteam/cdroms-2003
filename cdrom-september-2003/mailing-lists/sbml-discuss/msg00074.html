Hi All,

     Libsbml 1.0.1 is now available.

     This is a minor update for those who wish to use the latest version
of Xerces-C++ (2.2.0).  The previous version of the Xerces library (2.1.0)
was used to develop libsbml.  Version 2.1.0 did not use a C++ namespace.
Starting with version 2.2.0 a namespace was introduced.  The libsbml build
scripts will now detect either version (more specifically, either pre- or
post-namespace change) and compile accordingly.


Ben
