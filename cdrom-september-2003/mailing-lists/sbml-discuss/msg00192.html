Rational stoichiometries were specified in Level 1 for very good reason. I'd
like to know the difference that some found significant between a rational
and real number to be important enough to suggest removing the rational
stoichiometries. By loosing rational stoichiometries we loose some
advantages (significant advantages I think), by using the entire real number
set (within machine precision) some feel we gain more. The toss up between
the two depends on the balance of what we loose and what we gain.

Anyone know what these are, those in favour of floating point
stoichiometries should voice the advantages of the real number set over
rational numbers so that we can make a reasoned decision here. If there are
'real' advantages to using real over rational numbers then obviously we
should use them but lets make sure we know all the issues before we go for
one or the other.

There is one argument for using expressions for the stoichiometries, which
is the case when stoichiometries are a function of the state of the system,
happens sometimes with transport processes (known as slippage). If we do
(and we probably should) we will need to use MathML, sorry John, you might
have to have MathML after all in the stoichiometries (wish it could be
avoided).

Herbert Sauro

PS I agree with John that expressions in initial conditions would be nice.
