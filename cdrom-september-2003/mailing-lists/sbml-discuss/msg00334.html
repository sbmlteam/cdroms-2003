Normally I would say yes, but in this case I think the standard which 
many output devices use is defined in 1/72 of an inch.
So if we use the SI system this might complicate output and not offer 
any real advantage other than being the standard almost all over the 
world. E.g. the pt unit that is used to specify font sizes is defined 
as 1/72 of an inch so this time, inches might be the better choice.
