Hi Andrew and Michael,

we are currently working on several applications that aid in the 
visualisation of simulation results. Since we are using SBML as file 
format for the exchange of models and accompagnying information, it
appeared useful to us to include layout information (for biochemical
networks) in SBML. Herbert did something like that which was tailored 
for JDesigner. However, we wanted something more general which can be 
used by different applications and so, some people in my group have 
developed such an extension. A corresponding
description is attached for your information. We will make this
description available for download when people want to download the
respective visualisation software.

Of course, if you think, this is useful to other people in the 
consortium as well, you are welcome to make use of it.

Kind regards,

Ursula
