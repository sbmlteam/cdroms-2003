In cytoscape (http://www.cytoscape.org) we have an approach to layout, and
rendering which might be  useful to consider.   We also have a bare-bones
SBML reader.    

Layout, and the rendering of nodes, labels, and edges, are all
treated separately from the network model.  A new layout, or a new
visual style, can be designed at runtime, and saved for later use.
Since no single layout or visual style will always suffice, we set
out to make it easy to create new ones interactively, to save them,
and to use them again later.  

I'd be glad to create a java web start demo of this, using an SBML level 
1 model,
if anyone thinks it would be interesting.
