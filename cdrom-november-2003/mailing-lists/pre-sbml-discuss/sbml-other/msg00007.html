Nicolas 

IMHO SBML Level 2 doesn't need to identify species references.
so I don't see the need to add id or name attributes to
specieReference elements 

As Martin has already pointed out using ID and IDREF for 
ids is going to cause problems for modularity so
IMHO they should not be in Level 2.

IMHO we should keep discussion at this level of SBML off the sysbio list
and use this list instead.

Hopefully that's not to many HOs for one email :-)

yours Andrew
