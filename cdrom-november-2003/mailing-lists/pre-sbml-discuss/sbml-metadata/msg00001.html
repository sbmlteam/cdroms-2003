Hello!  I just wanted to pass on the link to the new working draft of
the CellML Metadata Specification:

http://www.cellml.org/public/metadata/index.html

This is tailored to CellML obviously, but the same format (and basic
concepts) would work for any XML-based language. The background section
of the spec. (http://www.cellml.org/public/metadata/background.html)
would be the most important read because it explains why we chose to use
the Resource Description Framework as the container for the metadata. 
If you want to find out more about RDF, check out
http://www.ilrt.bristol.ac.uk/discovery/rdf/resources/.  

The area of metadata would be a great place to start in making SBML and
CellML more compatible.  I'd love to hear any comments about all this. 
Also, if you have any questions, please don't hesitate to e-mail me.
