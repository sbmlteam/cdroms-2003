> > it is and will be free for academic and non-profit users?

> > works well as a description language, and it could be a useful standard

Nicolas:
> Now, you have to distinguish between "free" in the sens of "free beer"
> and "free" as in freedom. If you want your language to be used, you have
> to make it free according to the second meaning. 

Right.  And I completely agree with Drew Endy's argument:  as an E-Cell
architect I have decided to have strict criteria of 'OpenSource
interoperability' for system integration three years ago.  I'm always
asking myself when evaluating software for the system that 'if I choose
this for my software, and a user want to use this function, then what
happens?'

> Here are some example
> of "free" licences for languages.

> Python 2.0.1 license
>   http://www.python.org/2.0.1/license.html
> 
> Artistic license 2.0
>   http://dev.perl.org/rfc/346.html

More can be found here:
http://www.opensource.org/licenses/index.php


To me Artistic license is not an option (though 2.0 is better than the
original one).   I would choose and recommend to choose from one of
these families: (1) GNU GPL, or LGPL, (2) BSD-style (apache, MIT, ...),
(3) others such as Mozilla, Python).

Be careful that Python License is different from Python Software
Foundation License;  see the foreword of the Python Software Foundation
License.

> You can then make money on distribution of libs, progs and docs.

For properties that have characteristics where returns increase to scale
(software, industrial standards, music...), patent is a strategically
wrong idea if you want to make money by that.  Use the power of network
and copyright.  If the return decreases to scale, as in traditional
business models (selling jewelry, petroleum, dumb hardware..), patent
will empower you very much by protecting your *recipe*, but not the
product itself.

You would remember that copyright covers the material (or source code in
case of software) and patent is for concepts.  


Personally my opinion is the old good scientific tradition should be
preserved; scientific ideas should not be patented.
