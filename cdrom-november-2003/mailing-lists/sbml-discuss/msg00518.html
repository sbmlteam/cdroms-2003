Colleagues,

BioUML is free, open source, extensible Java framework for
systems biology. It spans the comprehensive range of
capabilities including access to databases with experimental
data, tools for formalized description of biological systems
structure and functioning, as well as tools for their
visualization and simulations.

The released new version of BioUML workbench supports SBML
format (version 1, level 1):
http://www.biouml.org/plugin_sbml.shtml?overview
http://www.biouml.org/download.shtml?0.7/release_notes

Additionally we have packed all models from SBML Model
Repository into BioUML module.
http://www.biouml.org/module.shtml?sbml_models

There is special forum at http://groups.yahoo.com/group/biouml
where you can post your comments or suggestions.

With best regards,
Fedor Kolpakov.
