ANNOUNCEMENT: SBML MODEL REPOSITORY
 
An SBML model repository has been set up, and can be found on
http://www.sbw-sbml.org/ModelsWebPages/ModelRepository.htm, or through
the links on the other pages of the Caltech ERATO Kitano Systems Biology
Workbench Development Group (http://www.sbw-sbml.org). 

Models are divided into two categories: models that have been published
in the peer reviewed literature, and unpublished models that are meant
to illustrate a particular concept. Each entry in the repository
contains a short description of the model, literature references (where
appropriate), and a link to the tool that has been used to generate the
SBML. The SBML code can be downloaded or viewed online.

The repository is maintained and curated by Maria Schilstra
(m.j.1.schilstra@herts.ac.uk). Instructions on how to submit models can
be found by following the link on the repository index page. At present
we can only accept SBML Level 1 models, but we hope to be able to accept
Level 2 models once the Level 2 definition has been frozen.
