Thank you. You're right, depending on the topic and level of abstraction of 
the model, it's not very simple. And we're not going converge anytime soon on 
labels for modifier types/classes for the general case.

So the only question that remains, I think, is whether any substantial subset 
of the current users are using SBML specifically for very low-level/granular 
models in which it _is_ simple, and a set of very primitive modifier labels 
would be all that's needed. If not, then obviously a simple set of labels 
would only be solving the wrong problem.

> The rate law:

>
> The species X behaves as an activator at low concentrations and an
> inhibitor at high concentrations. This rate law has been used in the past
> to model positive feedback.
>
> pH of course is a classic example (hydrogen ion) of an effector where
