Herbert

I don't believe its possible to parse a rate law to determine
if something is a modifier in the broad sense.
(Obviously if a species is in the rate law and isn't a reactant or
product then its a modifier.)

If we widen the definition of modifier so that modifiers can be
products or reactants then only humans can reliablely
classify species as modifiers or not.  However by incorporating this
wide definition SBML can capture information about all the
types of reactions that you and others have raised.  If
we use the narrow definition then we can't represent
all cases that exist.  

What does this mean in practice?:

It means that if some draws a diagram using the wide
definition of modifier then we can transmit that information
using the less restricted form of SBML.  If we have the
restricted form of SBML this can't transmitted:
the complete set of modifiers won't be present in the 
modifier lists.  We cannot expect a parsing diagram viewer
to decode the rate laws to reconstruct the complete
list of modifiers.

Thus I believe we need to have the list of modifiers.
