Hi All,

     libsbml 1.1.0 is now available on SourceForge:

       http://www.sourceforget.net/projects/sbml

     This release introduces several new features including:

       - A native Windows precompiled DLL,
       - A Developer's Manual (in Postscript and PDF formats),
       - XML Schema Validation,
       - Set vs. unset state tracking for fields, and
       - Parsing formula strings into abstract syntax trees.

     The next release of the libsbml, supporting the latest draft of
SBML L2, is almost complete and is planned for the end of this month (May).


Ben Bornstein
