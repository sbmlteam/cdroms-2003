MathSBML 2.1.4 has been released.
============================

MathSBML is an open-source package for working with SBML models in 
Mathematica. It provides facilities for reading SBML models, converting 
them to systems of ordinary differential equations for simulation and 
plotting in Mathematica, and translating the models to other formats.  
It supports both Level 1 and Level 2 SBML. This version contains fixes 
for a few bugs as well as a preliminary release of an SBML Model 
Builder that allows one to manually construct SBML models from 
Mathematical expressions.

The new version can be found at http://www.sf.net/projects/sbml

More detail, including full on-line documentation for all functions as 
well as examples,  can be found at http://www.sbml.org/mathsbml.html

Summary of Major Changes
======================

Version 2.1.4
=============
1. Fixes several errors in predefined rate laws used in SBML Level 1.
2. Adds several Model Builder Functions: functionToSBML, speciesToSBML, 
reactionToSBML.

Version 2.1.3
=============
1. Fix bug in SBMLWrite/XPP for level 1 files, initialization of 
SBMLFunctions.
2. Fix bug in SBMLWrite/Fortran, variable name substitution in Jacobian.
3. Level 1 models place units in their own namespace (already done for 
level 2).
4. Automated Solve/Plot in SBMLRead
5. Option passing to SBMLCopy

Version 2.1.2
=============
1. Fix bug that causes system to ingore objects with embedded 
underscores
("_" character) in MathML expressions.
2. Function SBMLCopy that copies & prettyprints a file.

Version 2.1.1
=============
Version 2.1.0 was inadvertently named 2.1.1 internally. To avoid 
confusion,
there is no separate version called 2.1.1.

Version 2.1
============
1. Fixes several bugs identified at the 1st SBML Hackathon.
2. SBMLWrite supports XPP, HTML and Fortran Formats.
3. Includes all pre-defined rate functions for SBML Level 1.

Version 2.0.
=============
1. SBMLRead is supports SBML Level 2 as well as Level 1.

Version 1.1
===========
1. All variables  placed into context given by name of model.
2. Various bug fixes.

Version 1.0
===========
Original Release
