Dear all,

As part of the sbml.org site redesign, we've added a
full-featured web interface for sbml-discuss.  The software
we're using allows bidirectional gating to/from a mailing
list, and postings made in one automatically appear in the
other.  This allows people to use their preferred kind of
interface (web or email) to interact with sbml-discuss.

Important: Everyone who has ever posted to sbml-discuss
already has an account on the web interface, but no password
yet.  Before you can post using the web interface, you must
set a password for your account.  To do this, use the
'Login' button, type in your login (which is your email
address without the host part -- e.g. "mhucka"), and click
on "Forgot password".  You'll be emailed instructions.
(Note that this is not necessary if you only use the mailing
list directly -- you only need a password if you want to
post from the web interface.)

There will undoubtedly be some glitches as we discover how
to make everything work nicely, so please be patient if
things don't seem to work smoothly right at the beginning.
And do let us know when you encounter problems and oddities.

MH
