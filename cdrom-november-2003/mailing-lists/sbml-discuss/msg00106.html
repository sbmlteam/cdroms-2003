Dear SBML enthusiasts,

The MathSBML package for Mathematica is now available.  This
package provides facilities for reading models expressed in
SBML Level 1, converting them to systems of ordinary
differential equations, and solving and plotting the results
in Mathematica.

Some feature highlights in this release include:

- SBMLRead: function that reads an SBML file and translates
   it into a Mathematica data structure consisting of
   differential equations, initial conditions, variables and
   replacement rules

- SBMLNDSolve: function that solves the system of
   differential equations produced by SBMLRead using
   Mathematica's NDSolve.

- SBMLPlot: function that generates a plot of the resulting
   solutions.

- Documentation in the form of a README file as well as
   function documentation within Mathematica itself.

MathSBML is available as a downloadable ZIP archive from the
SBML project area on SourceForge,

   http://sourceforge.net/projects/sbml/

It is also available directly from the public CVS repository
hosted on SourceForge; information on accessing this is
available from the URL above.

For all those interested, I look forward to your feedback.

Bruce E. Shapiro
