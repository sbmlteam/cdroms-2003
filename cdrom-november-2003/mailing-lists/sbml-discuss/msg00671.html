Greetings,

BioPAX - Biological Pathway Exchange - http://biopax.org

BioPAX has been created to enable sharing of pathway information, such
as signal transduction, metabolic and gene regulatory pathways, using a
standard data exchange format.

We are pleased to announce the release of BioPAX Level 1, version 0.5
(draft release).  This release provides a preview of the BioPAX pathway
data exchange format in order to generate discussion and feedback from
the community, which we will use to refine the ontology.  Large changes,
based on community feedback, may be made before the next draft is
released.

BioPAX ontology files and documentation are freely available (under the
GNU LGPL license) here: http://www.biopax.org/Download.html

We request feedback on BioPAX via the biopax-discuss mailing list, which
you can join here: http://www.biopax.org/mailman/listinfo/biopax-discuss

Issues that are raised on this list will be tracked using SourceForge in
the BioPAX bug tracker here:
http://sourceforge.net/tracker/?atid=575904&group_id=85345&func=browse

Thanks for your time and we look forward to hearing from you.

Best regards,
BioPAX Core Group
