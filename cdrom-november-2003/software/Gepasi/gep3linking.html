Guidelines for adding links to Gepasi on web pages

You are most welcome to link your pages to this web site! Please follow these guidelines when doing so:

  1. Add a link in one of those two formats:
    • Text link:
      • Add this HTML code to your page:
        <A HREF="http://www.gepasi.org">Gepasi: computer simulation of biochemistry</A>
      • The link will look like this in your page: Gepasi: computer simulation of biochemistry
    • Graphical link using the banner:
      • Download Gepasi's banner from here and place it in the same directory as your HTML file.
      • Add this HTML code to your page:
        <A HREF="http://www.gepasi.org"><IMG BORDER=0 HEIGHT="40" WIDTH="400" SRC="gepbann2.png" ALT="Gepasi - computer simulation of biochemistry"></A>
      • The link will look like this in your page:
  2. Email the webmaster informing him of this new link to the Gepasi web site.
