The end of an era...

There are no more plans to release new versions of Gepasi. Version 3.30 should be the final release (barring discovery of some serious bug in this version).

... and the dawn of a new one

But we are now working hard on a successor of Gepasi, named COPASI. This is a collaboration between the Biochemical Networks Modeling Group (the Mendes lab) and a group at the European Media Lab, Heidelberg, led by Ursula Kummer.

COPASI will have the following exciting features:

  • Reads Gepasi files (backwards compatibility)
  • Runs on Solaris, Windows, Linux, and Mac OS X
  • Capable of carrying out stochastic integrations, in addition to solution of ODEs (Gepasi's method)
  • Non-linear dynamic analyses, such as bifurcation analysis
  • Other nice features (watch this space!)
