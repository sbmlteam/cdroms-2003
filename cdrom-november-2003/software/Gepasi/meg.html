MEG - Model Extender for Gepasi
home page

Features

  • MEG is a program for building models of differentiated and heterogeneous biochemical systems.
  • Models built with MEG can be run on the metabolic simulator Gepasi.
  • With MEG you can set up models of liquid cultures, 2-dimensional tissues and 2-dimensional tissues immersed in a liquid medium

Download

MEG is available to run under:
Windows 95 and above on Intel computers meg-i.zip
Windows NT on Alpha computers meg-a.zip

Installation

As of Gepasi release 3.30, MEG is included with the main package. If you have an older version of Gepasi, then you should download and install the new version.

If you do not want the new version (but you should!), unzip the ZIP file above into the directory where you have installed Gepasi.
