MathSBML Installation Instructions

  1. Unpack the MathSBML archive by

    1. Click here to open the ZIP archive included on this CD-ROM.
    2. Unpack the archive and save it on your computer's hard disk.


  2. Install MathSBML

    1. Locate the file mathsbml.m in the unpacked archive.
    2. Copy mathsbml.m to one of the following locations:
      • Your home directory. In most multi-user systems this will be the directory you are in when you first log onto the system. The location varies in earlier versions of Windows or Mac OS. To determine this directory, open Mathematica and enter $HomeDirectory (Mathematica 4.x, Details) or $BaseDirectory (Mathematica 5.x, Details)
      • The Mathematica Add-Ons Application subdirectory. A typical location for this directory is ~/Library/Mathematica/Applications. To find out where Add-Ons directory is located, open Mathematica and enter $AddOnsDirectory (if you want the file to be visible to all users on your system) or $UserAddOnsDirectory (if you want the file to only be visible to you). There should already be an Applications sub-directory within this directory.
      • The Mathematica Add-Ons AutoLoad directory. A typical location for this directory is ~/Library/Mathematica/AutoLoad. This is the AutoLoad subdirectory of the $AddOnsDirectory. Click here for details on AutoLoading packages.
      • Any directory listed when you enter $Path during a Mathematica Session (Details)
      • Any other directory you prefer to keep it in.
    3. If Mathematica is currently open, quit Mathematica.
    4. Restart Mathematica
    5. If you installed mathsbml.m in
      • The AutoLoad directory - MathSBML will automatically load when you launch Mathematica.
      • Your Home Directory, the Add-Ons Application Directory, or any of the directories listed by the Mathematica $Path variable, load MathSBML by entering 
        >>mathsbml.m
      • Any other directory, load MathSBML by entering 
        >>"path/mathsbml.m"
        where path is the full path name of the directory where mathsbml.m is located.
    6. Click here fore detailed information on loading packages.


  3. Additional details on Mathematica can be found in the following locations:

    • Version 5
    • Version 4
    • Earlier Versions (Note: MathSBML will not work with versions of Mathematica prior to version 4).
