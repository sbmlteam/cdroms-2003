MathSBML Model Builder: SBMLGridPlot

SBMLGridPlot[solution,options]
SBMLGridPlot[solution,{var1,var2,...}, options]
SBMLGridPlot[solution,{var1,var2,...}, {tbegin,tend},options]

solution is the output of SBMLNDSolve

These functions will plot each specified variable (or the entire solution set) for the specified times (or the entire time span) on a separate graph, with the top of the graph labeled with the variable name. The plots will be arranged on a grid with three plots per line. The context of the variables should not be included in the list of variable names.

Options include:

columns->3, number of plots per line

Any valid option for Plot may also be used.
