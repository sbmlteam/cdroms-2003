MathSBML Model Builder: newModel

newModel[options] resets counters and various internal variables that are used by the modelBuilder. The current model is completely erased.

Options

Option Default Description
id SBMLModel Value of id field of model.
name none Value of name field of model. If not specifiec, will be the same as id.
