MathSBML Model Builder:createModel

createModel[] returns the sbml for the given model.

createModel[filename] writes the sbml for the given model to the specified file.

createSymbolicModel[] retursn symbolic sbml for the given model.

createModel, createSymbolicModel are new in 2.1.5.
