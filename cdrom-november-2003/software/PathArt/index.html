PathArt is a comprehensive collection of curated data from literature for more than 400 signaling and metabolic pathways. It provides a collection of high priority disease information with an emphasis on candidate and pathway responsive genes. PathArt displays pathways in a dynamic way and includes modules to analyze microarray data.

The Database component integrates pathway information curated from peer-reviewed articles, patents and other public domain sources, that enables discovery scientists to:

 
Identify & Compare pathways across physiologies/ diseases/ organisms
·
Finding rate limiting steps in a pathway
 
Analyze microarray data using sequence/probe ID based searches & in time series
 
Identify disease pathways and the critical disease genes
  ·
 View & build information on cross talking pathways
 
Dynamic Pathway Articulator Component is a web-based graphical visualization tool that analyzes the underlying database and dynamically builds a pathway schematic. The molecules are displayed in a cell schematic and displays annotated information on bio-molecules and interactions between them.
 

Microarray Data Analysis Component provides information on pathway responsive genes summarized from over ten public domain databases. Analysis on genes is provided on tissue specificity, distribution on chromosomes, molecular function and involvement in biological function.
 
Data Retrieval and query capabilities allow searching the database on fields such as organism, disease, tissue etc. The gene expression profiler maps the genes onto their relevant pathways and allows filtering of over-expressed and under-expressed genes.

Technical specifications
Highly robust web-based application with Oracle backend and a Java front end - can be viewed using any standard browser.
 
Runs on Windows NT, Windows 2000, Linux and Solaris platforms.
 
Data interface provided in XML format, and allows client database integration.
 
Applications
The database would be an invaluable tool in the process of target identification and validation, discover regulatory mechanisms and to study kinetics of small molecules.
