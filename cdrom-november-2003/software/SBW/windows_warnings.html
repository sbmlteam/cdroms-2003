Known Issues with the SBW Installer Under Windows

We are aware of the following issues with the Windows installation of SBW.

  • Windows 98SE, 2000 and XP are the only versions supported at this time

    SBW has not be designed or tested to run with versions of Windows other than 98SE, 2000 and XP. It has been the case that users who attempted to install SBW on other versions of Windows have encountered problems; therefore, we do not recommend attempting to use SBW with Windows versions other than 98SE, 2000 and XP. We apologize if your version of Windows is not in the supported list, but our limited resources prevent us from supporting more systems.

  • Conflict with ZipMagic

    The SBW Installer is built using ZeroG's InstallAnywhere application. InstallAnywhere is known to be incompatible with systems running OnTrack Software's ZipMagic compression utility. Users with ZipMagic installed will need to disable, or uninstall ZipMagic in order to use InstallAnywhere installers, including the SBW installer.

    If you attempt to run the SBW installer on a system that has ZipMagic installed and running, the installer will fail with the following error message:

 	at java.io.RandomAccessFile.

    For more information, please support item #282 in the Zero G support knowledge base.

  • SBW does not operate properly under Windows 98

    SBW 1.0.2 appears to exhibit problems running under Windows 98. Windows 98SE does appear to work, but plain 98 has not be tested by the SBW developers and users who have tried it have reported failures to run.

If you experience problems not described here, please report them using our bug reporting and tracking facilities.
