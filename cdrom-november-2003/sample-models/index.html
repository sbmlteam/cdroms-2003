Sample Models in SBML Format

Models from the SBML Repository

The SBML model repository at the SBML project website (http://www.sbml.org) contains a number of models that have been expressed in SBML. A sample of those models is included here. For detailed information about each model, click on its hyperlink in the Model Description column of the table below. This will bring up a short description of the model, links to publications in which it is described, the name of the tool that was used to generate the SBML, and the name of the creator of the model file. Zip files containing model definitions in SBML and other formats can be downloaded from the page containing the model description. (Warning: some web browsers, including Safari under MacOS X, will not display anything if you attempt to load an XML file; in those browsers, you must use the Zip file.)

Most models were validated by importing them into a simulator that reads and interprets SBML, and performing a simulation using the parameter and boundary values specified in the SBML file.

SBML Test Suite

The SBML Test Suite is a collection of models and associated files intended to serve as a test set for developers of software with SBML input/output capabilities. We are still in the process of developing this test suite, and at present, documentation is sparse.

The suite is currently a collection of files in plain-text (UTF 8) format. There is no installation other than unpacking the archive of files. Click on one of the following links to obtain the archive:

  • SBML Test Suite in Zip archive format
  • SBML Test Suite in gzip'ed Unix tar format
